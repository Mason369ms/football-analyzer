## 🎯 常见问题

### Q1: 仪表盘启动后看不到序号列？

**原因**：可能运行的是旧代码

**解决**：
```powershell
# 1. 停止仪表盘（Ctrl+C）

# 2. 确认 PYTHONPATH 设置正确
$env:PYTHONPATH='src'

# 3. 重新启动
python -m football_sim.cli dashboard --server fastapi --port 8766

# 4. 清除浏览器缓存
# Chrome: Ctrl+Shift+Delete → 清除缓存
# 或使用无痕模式: Ctrl+Shift+N
```

### Q2: 近期分析仍然有重复？

**原因**：可能未清理历史数据

**解决**：
```powershell
# 运行清理脚本
$env:PYTHONPATH='src'
python scripts/clean_duplicates.py

# 输入 'y' 确认清理

# 重启仪表盘
python -m football_sim.cli dashboard --server fastapi --port 8766
```

### Q3: 序号显示为"?"或为空？

**原因**：赛事数据可能未包含 round_info

**解决**：
```powershell
# 重新抓取赛事数据
$env:PYTHONPATH='src'
python -m football_sim.cli fetch-matches --date 2026-06-04

# 重启仪表盘
python -m football_sim.cli dashboard --server fastapi --port 8766
```
